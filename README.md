# bitarray
A simple bitarray library for nim.

# Installation
```
git clone https://github.com/YesDrX/bitarray.git
cd bitarray
nimble install
```
or
```
nimble install nim-bitarray
```

# [Documentation](https://yesdrx.github.io/bitarray/)

# Example
```nim
import bitarray
when isMainModule:
  var
    a = newBitsArray(70)
    b = newBitsArray(70)
  
  echo a
  echo b
  a.setBits(0,1,2)
  b.setAll
  echo a & b
  echo a | b
  echo a ^ b
  echo ~a
  echo a.nbytes
```
